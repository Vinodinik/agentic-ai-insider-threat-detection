AN AGENTIC AI FRAMEWORK FOR INSIDER THREAT DETECTION USING MACHINE LEARNING APPROACH AND RULE-BASED DECISION SYSTEMS IN USER BEHAVIOR ANALYTICS USING INTERACTIVE FLASK DASHBOARD

Insider threats have become a major cybersecurity concern in modern organizations, where authorized users may misuse their access to compromise sensitive data and systems. This project proposes an Agentic AI–based Insider Threat Detection framework using User Behavior Analytics (UBA) to proactively identify and mitigate abnormal user activities. The system processes enterprise user activity logs through a structured pipeline that includes data preprocessing, anomaly detection using the Isolation Forest algorithm, and simulation of synthetic threats to enhance robustness. Detected anomalies are further analyzed using a reasoning module to provide contextual classification, followed by a risk scoring mechanism that assigns normalized scores and severity levels to prioritize threats. The system also includes automated response recommendations and an explainability layer that generates human-readable insights for better understanding and decision-making. Implemented as a Flask-based interactive dashboard, the solution features an interactive dashboard for real-time monitoring and visualization, along with automated report generation, providing a scalable and intelligent approach to insider threat detection and management.
